# 小绿书（微信图片帖）设计规范

## 平台特性

微信图片帖（小绿书）是在公众号内以图片序列展示的内容形式，最多 20 张图片。用户通过左右滑动浏览，完读率由每张图的视觉吸引力决定。

**与小红书的关键差异**：
- 受众：微信用户整体偏成熟，设计感接受度更高
- 浏览方式：横向滑动（vs 小红书的竖向滚动）
- 分享场景：朋友圈/微信群（更注重分享动机）

---

## 封面规范

- 比例：3:4 竖版
- 单一主题，文字 ≤25%
- 清晰传达内容价值主张
- 字体：可以比小红书更设计感，但避免过于沉闷的商务风

### 封面内容类型与设计

| 内容类型 | 视觉风格 | 典型构图 | 配色建议 |
|----------|----------|----------|----------|
| 旅行图集 | 真实摄影风 | 风景大图 + 小字标题 | 自然色温，暖调 |
| 好物测评 | 产品展示风 | 居中产品 + 环境氛围 | 纯净背景，品牌色点缀 |
| 知识干货 | 信息图式 | 结构化布局 + 数据可视化 | 克制冷色调 |
| 日常打卡 | 生活感照片 | 随拍构图 + 氛围感 | 暖色调，胶片质感 |
| 食谱教程 | 食物特写 | 俯拍/45度角 | 暖调，食材原色 |

---

## 内容图规范

### 密度

- 每张 ≤3 个核心信息点
- 文字占比 20-40%
- 留白 ≥30%

### 视觉一致性

- 色调在整个图集中保持统一
- 字体风格保持一致
- 构图风格保持一致（勿混用极简和复杂风格）

### 多样性

- 不同页背景/场景不重复
- 构图角度有变化
- 至少交替使用 2 种以上构图类型

### 禁止

- 与封面风格断裂
- 文字过密难以扫读
- 纯色大背景（除非整体为 minimal 极简风）

---

## 尾图规范

尾图服务转化，是用户看完后决定是否互动的关键。

### 尾图类型

| 类型 | 适用内容 | 设计要点 | 示例 |
|------|----------|----------|------|
| `follow` | 知识干货 | 提示关注获取更多 | "关注我，每周分享XX干货" |
| `comment` | 测评对比 | 引导评论区讨论 | "你用哪个？评论区告诉我" |
| `traffic` | 种草推荐 | 引导分享/收藏 | "转发给需要的朋友" |

### 尾图设计

- 保持与封面和内容图一致的视觉风格
- 大面积留白，信息极简
- 一句话 + 一个行动引导
- 不要堆砌多个 CTA

## 6 种布局模式

### 1. 编号清单

适用：步骤、方法、排行榜
```
1️⃣ 第一步标题
   简短说明

2️⃣ 第二步标题
   简短说明

3️⃣ 第三步标题
   简短说明
```

### 2. 对比双栏

适用：优缺点、前后对比、产品选择
```
┌──────────┬──────────┐
│   选项A   │   选项B   │
│  优点...  │  优点...  │
│  缺点...  │  缺点...  │
│  适合...  │  适合...  │
└──────────┴──────────┘
```

### 3. 步骤流程

适用：教程、操作指南、制作过程
```
Step 1 ──→ Step 2 ──→ Step 3
 准备      操作      完成
```

### 4. 标注图解

适用：产品细节、知识点标注、位置说明
```
┌──────────────────┐
│    主图/产品      │
│  ① 特点说明      │
│     ② 另一个特点  │
│  ③ 第三个特点    │
└──────────────────┘
```

### 5. 数据卡片

适用：测评数据、排名、统计
```
┌─────────────────┐
│   指标名称       │
│   9.5 / 10      │
│ ████████████░░  │
│   简短评语       │
└─────────────────┘
```

### 6. Q&A 对话

适用：FAQ、知识解答、常见误区
```
Q: 常见问题？
A: 解答内容...

Q: 另一个问题？
A: 另一个解答...
```
